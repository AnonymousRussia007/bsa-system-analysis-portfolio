# Документация портфолио

В этом разделе собраны вспомогательные документы
для навигации по портфолио системного аналитика.

- [Каталог артефактов](catalog.md) — сводная таблица кейсов,
  навыков и итоговых материалов.
- [Проверка репозитория](validation.md) — что сохранено,
  что удалено как учебная обвязка и какие ограничения остаются.

Основные кейсы находятся в папке [cases/](../cases/).
