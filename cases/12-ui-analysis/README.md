# Анализ пользовательских интерфейсов

## Кратко

Спроектирован набор экранов для менеджера, мастера и клиента.

Определены сценарии, данные, операции, проверки,
сообщения об ошибках и элементы управления.

## Что демонстрирует

- перевод требований в пользовательские сценарии;
- формирование каталога экранных форм;
- проектирование полей, операций и проверок;
- подготовку связанных макетов интерфейсов.

## Основные артефакты

- `artifacts/ex00_product_presentation.md` — выбор формы продукта;
- `artifacts/Ex01/ex01_references_screen_forms.md` — формы справочников;
- `artifacts/ex02_references_data.md` — состав данных и проверки;
- `artifacts/ex_03_key_scenarios.pdf` — ключевые сценарии ролей;
- `artifacts/ex04_screen_forms_list.md` — каталог экранов;
- `artifacts/Ex05/ex05_screen_forms_wireframes.md` — спецификация 22 макетов;
- `artifacts/ex_06_controls.xlsx` — элементы управления и правила;
- основной фокус — каталог экранов, спецификация макетов и таблица controls.

## Статус

Учебный командный кейс. Макеты не являются реализованным приложением.
