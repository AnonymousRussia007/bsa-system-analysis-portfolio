# Анализ заинтересованных сторон

## Кратко

Для сервиса доставки выделены заинтересованные стороны, их роли, интересы, потребности и проблемы.

## Что демонстрирует

- идентификацию стейкхолдеров;
- анализ влияния и ожиданий;
- построение луковичной диаграммы;
- развитие глоссария предметной области.

## Основные артефакты

- `artifacts/ex00_dlv_stakeholders.xlsx` — реестр заинтересованных сторон;
- `artifacts/ex01_dlv_onion.png` — луковичная диаграмма;
- `artifacts/ex02_dlv_needs.xlsx` — интересы, потребности и проблемы;
- `artifacts/ex03_dlv_glossary.xlsx` — глоссарий;
- `artifacts/Проект Стейкхолдеры.docx` — сводное описание;
- основной фокус — луковичная диаграмма, реестр и матрица потребностей.

## Статус

Учебный кейс. Формального согласования с реальными стейкхолдерами нет.
